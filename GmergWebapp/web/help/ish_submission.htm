The ISH Submission Details page.

This page displays detail of a particular ISH submission in the database

http://gudmap.hgu.mrc.ac.uk/gudmap/pages/ish_submission.html?id=GUDMAP:1

www.euregene.org/euregenedb/pages/ish_submission.html?id=ERG:1

It is displayed by clicking on a :ID in the Browse ISH Data page. 

Data is displayed in the following sections:

  • ID
  • Gene symbol.
  • Stage.
  • Original assay images.
  • Authors.
  • Principal investigator.
  • Submitter.  
  • Expression mapping.
  • Probe details.
  • Specimen details.
  • Linked Publications
  • Linked Submissions
  • Acknowledgements

ID
The accession number for this entry.

Gene symbol
Links to the Gene Details page.

Stage.
Links to staging information.

Original assay images.
Thumbnails of all the original images submitted. Only one thumbnail is displayed in the browse all list. This is the first image that is referenced in the submitted data. From the Submission Details page all the original images can now be viewed as thumbnails. Clicking on each image will display the original sized image plus supplementary data in its own window.

Authors.
The authors associated with the entry.

Principal investigator.
The PI for the submitting lab

Submitter.
The person who submitted the data.  

Expression mapping.
Gene expression and annotation displayed in an antomical tree structure. See Expression for the key to the expression mapping. The tree can be collapsed or expanded by clicking on the + or - icons at each node. Clicking on the component will pop up a new window with annotation and component details.

Probe details.
The Probe section of this page contains three external links. The first of these is “Probe MGI Acc ID” and links to MGI database details on the specific probe. The second external link is labelled Genbank ID and take the user to NCBI's genbank database and provides additional information on the sequence.

Specimen details.
Other staging system, Strain, Genotype, Specimen preparation, Fixation and Embedding methods are all provided here.

Linked Publications
Publications in which the current data is referenced.

Linked Submissions
Other submissions in this and other databases to which this entry is linked.

Acknowledgements
Contact details and reasons for those being acknowledged.
