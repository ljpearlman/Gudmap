The Focus page.

This page is divided into headings which represent abstract anatomical groupings which are likely to be of most regular interest.

These groupings are themselves grouped according to age.

  • Up to E11.5
  • After E11.5
  • Adult

At the moment these anatomical groupings are stated in text format but we would like to replace these text terms with appropriate images.

The user is able to see the list of components which constitute the group and the number of corresponding entries in the database for that group.

By clicking on the group name or a member of that group, the relevant submissions will be returned in the format of the Browse All (ISH) page.

Currently the data presented in this page and subsequent links from the page apply only to ISH data. Microarray data has still to be incorporated into the result sets.

 

At the moment this page is under development but will be available in the near future.
