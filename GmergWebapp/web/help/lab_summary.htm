The Lab Summary page.

This page contains submissions details for each of the labs involved in the Gudmap project. The results are grouped by Lab.

 http://gudmap.hgu.mrc.ac.uk/gudmap/pages/lab_summaries.html

Alongside the PI for each lab there are links to view All the ISH or Microarray entries for that lab with a date for the last entry from that lab to the database.

Details for each lab include:

  • The Submission Date.
  • The number of submissions
  • The assay type
  • The status of the submissions.

The status of submissions is either ‘Available’ ie (publicly available in the database) or ‘Editing in Progress’ . (ie. The total submitted for a dataset on a specific date minus the ‘Available’ entries for that same dataset).

Clicking on the 'number of submissions' for a row with 'Available' status, will display those entries in either ISH or Microarray Browse All format.
