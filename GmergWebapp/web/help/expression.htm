Expression

The anatomy tree is used to display gene expression details in anatomical components.

uses 2 levels of expression included in submissions to the database:

  1. Present
  2. Not Detected

The default for all components is 'Not Examined'

The primary annotated component is displayed in the treee in UPPERCASE.

If expression is 'Present' the color (red) is inferred upwards in the tree hierarchy.

If the expression is 'Not Detected' the color (blue)  is inferred on components downwards in the tree hierarchy.

The default layout of the tree in the Submission Details page is to hide 'Inferred Not Detected'.

Clicking on the + or – icons to the left of the tree will expand/collapse that node.

Clicking on the folder or page icon associated with a term in the tree (immediately to the left of the term) will pop up a window with anatomy and annotation details.

The EMAP:ID is displayed at the end of the string containing the term.

EMAP Id's are generated by the Mouse Atlas Anatomy Database at the HGU in Edinburgh.
