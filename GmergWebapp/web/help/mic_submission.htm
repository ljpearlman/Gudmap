The Microarray Submission Details page.

This page displays detail of a particular ISH submission in the Gudmap database

It is displayed by clicking on a GUDMAP:ID in the Browse Microarray Data page. 

http://gudmap.hgu.mrc.ac.uk/gudmap/pages/mic_submission.html?id=GUDMAP:2168

Data is displayed in the following sections:

  • GUDMAP:ID
  • Theiler Stage
  • Supplemental Data files
  • Gene List
  • Principal Investigator
  • Submitted to GUDMAP By
  • Sample Details
  • Series Details
  • Platform Details
  • Specimen

 

  • GUDMAP:ID
    The GUDMAP accession number for this entry.
  • Theiler Stage
    Links to the Theiler staging information at HGU
  • Supplemental Data files
    • CEL file (click to save to disk)
    • CHP file (click to save to disk)
    • RPT file (click to save to disk)
    • EXP file (click to save to disk)
    • TXT file click to view
  • Gene List.
    This is presented as an embedded window.
    • Select All | Deselect All links at top left allow the user to quickly select / deselect all the rows on the page. (A selected row has a tick in the Select column
    • Navigation throught the data is by use of the navigation arrows (first,next,previous,last), or by typing a valid integer into the 'Go to page' text box and clicking the GO button.
    • Table of array data with the following columns
      • Select
      • Gene Symbol (link to gene details page) (sortable)
      • Probe ID (sortable)
      • Signal (sortable)
      • Detection (sortable)
      • P-Value (sortable)
    • Scroll to the bottom of the gene list table where there are two options.
    • (Left cell) Add to My collection with Selected genes (default)
      • Options from the ‘Add To’ dropdown are ‘Replace, Get Intersection With, and Get Difference With’
      • Options from the ‘Genes’ dropdown are ‘Genes’
      • The GO button performs the action
    •  (Right cell) View my Gene collection. -  Click to view current collection.
    • When viewing the current collection, the number of ISH entries  and number of Microarray entries for each gene will be displayed after every gene symbol.
  • Principal Investigator
    Name, address and email of the PI
  • Submitted to GUDMAP By
    Name, address and email of the submitter
  • Sample Details
    • Sample GEO ID : link to GEO website for current sample
    • Title
    • Source. Eg Testes
    • Organism. Mus musculus
    • Strain
    • Mutation
    • Sex (M=Male, F=Female, U=Unknown, BOTH
    • Developmental age. Eg pnd 2
    • Theiler Stage. Eg 28
    • Dissect Method
    • Molecule
    • A_260_280
    • RNA extract protocol
    • Target amplification manufacturer/kit
    • Target amplification protocol
    • Rounds of amplificaton
    • Amount labelled Target Hybridized to Array
    • Label
    • Array Hyb/Wash/Scan Protocol
    • GCOS Tgt Value
    • Data Analysis Method
    • Reference Used
  • Series Details
    • Series GEO ID : link to GEO website for current series
    • Number of samples (link to display number of samples)
    • Title
    • Summary
    • Type
    • Overall Design
  • Platform Details
    • Platform GEO ID: link to GEO website for current platform
    • Title
    • Name
    • Distributiion
    • Technology
    • Organism
    • Manufacturer
    • Manufacturer  Protocol
    • Catalogue Number
  • Specimen
    • Theiler Stage (link to HGU)
    • Other Staging System. Eg pnd 2
    • Strain
    • Genotype (wildtype/mutant)
    • Specimen Preparation
      • Fixation method
      • Embedding
