The Browse All (ISH) page.

This page displays details of all ISH submissions in the database. The default sort is by gene.
 http://gudmap.hgu.mrc.ac.uk/gudmap/pages/ish_browse_all.html

http://www.euregene.org/euregenedb/pages/ish_browse_all.html

This page will only show the ISH submissions which are ‘Available’ in the database.  The number of submissions currently with ‘Editing in progress’ status from each lab will be shown in the Lab Summary page.

Selection
There are 3 options for selection

  1. Select All. This selects all the rows in the current page by checking the select box for each row.
  2. Deselect All. This deselects all rows which in which the select box is checked.
  3. Show Selected in New Window. Displays all the currently selected rows in a new window.

Page navigation
The default number of entries shown per page is 20. This can be changed by selecting a different value from the dropdown box at the top right of the page. 'Display n Entries Per Page'.

The total number of pages available is shown at the top left of the page as: 'Page x of n'.

Navigation through the pages is by the set of 4 arrows. These arrows represent from left to right:

  • First Page, Next Page, Previous Page, Last Page.

If the user chooses to display all results in a single page, the “Go to Page” and navigation arrows will no longer be available.

Alternatively the user can go straight to a page number by typing in a valid integer into the 'Go to page' box and either pressing 'Return' key or clicking the 'Go' Button.

Column Headers
Each entry is arranged in tabular format and columns contain from left to right:

  • Select
  • ID
  • Gene Symbol
  • stage
  • Age
  • Lab
  • Submission Date
  • Assay Type
  • Specimen Type
  • Thumbnail

Sorting
With the exception of Select and Thumbnails, the page can be sorted according to a specified column by clicking on column header.

For example if the user wants to sort each entry by :ID they would click the “ID” link at the top of the column. This would order the rows as :1, :2, :3, etc. Clicking on the same column header again would sort the rows in reverse order of ID.

Additional Links

From each row there are links to additional pages relating to the entry. The accession number links to the specified Submission Details Page and the gene symbol links to the Gene Details Page.

Clicking on the thumbnail will show the orginal image in its own window with accompanying supplementary data.
