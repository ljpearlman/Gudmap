The help pages refer to the view of the database and will subsequently be referred to as the 'database site'.

The help pages are intended both as a user guide and a test case suite.

All help pages contain a navigation bar with links to 'FAQ', 'Pages' and 'Database'.

  • FAQ. A list of frequently asked questions. Visiting this page first is recommended as the first step to answering questions regarding the functionality of the database site.
  • Pages. A list of help pages directly related to views of the database as presented on the database site.
  • Database. A link back to the main entry page for the database site.

The website is divided into two parts.

  1. Information about including Research and Resource topics. Hosted at Harvard.
  2. The view of the database containing ISH and Microarray experiment data. Hosted at HGU, Edinburgh.

 

Entry to the database site is either

    • via the new website (www.gudmap.org) and click on ‘Database’
    • directly @ http://gudmap.hgu.mrc.ac.uk/gudmap

Entry to the database site is via the web site (www.euregene.org/euregenedb/) (www.euregene.org/xenopusdb/)

On each page in the database site, above the title ( Gene Expression Database) is the database navigation bar. Clicking the links in this bar will allow the user to navigate to the following pages within the database site.

  • Browse All (ISH)
  • Browse All (Microarray)
  • Focus
  • Query
  • Lab Summary

Navigation
All pages within the database site will have the same navigation bar to access different areas of the database site.

Footer
All pages within the database site will have a footer with links to 'Future Developments', 'Editor login' and 'Contact details'.
