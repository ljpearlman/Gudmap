The Browse All (Microarray) page.

This page displays details of all Microarray submissions in the Gudmap database. The default sort is by GUDMAP:ID
http://gudmap.hgu.mrc.ac.uk/gudmap/pages/mic_browse_all.html

This page will only show the microarray submissions which are ‘Available’ in the database.  The number of submissions currently with ‘Editing in progress’ status from each lab will be shown in the Lab Summary page.

Selection
There are 3 options for selection

  1. Select All. This selects all the rows in the current page by checking the select box for each row.
  2. Deselect All. This deselects all rows which in which the select box is checked.
  3. Show Selected in New Window. Displays all the currently selected rows in a new window.

Page navigation
The default number of entries shown per page is 20. This can be changed by selecting a different value from the dropdown box at the top right of the page. 'Display n Entries Per Page'.

The total number of pages available is shown at the top left of the page as: 'Page x of n'.

Navigation through the pages is by the set of 4 arrows. These arrows represent from left to right:

  • First Page, Next Page, Previous Page, Last Page.

If the user chooses to display all results in a single page, the “Go to Page” and navigation arrows will no longer be available.

Alternatively the user can go straight to a page number by typing in a valid integer into the 'Go to page' box and either pressing 'Return' key or clicking the 'Go' Button.

Column Headers
Each entry is arranged in tabular format and columns contain from left to right:

  • Select
  • Gudmap ID
  • GEO Sample ID
  • Theiler stage
  • Age
  • Lab
  • Submission Date
  • Assay Type
  • Tissue Type

Sorting
With the exception of Select and Thumbnails, the page can be sorted according to a specified column by clicking on column header.

For example if the user wants to sort each entry by Gudmap:ID they would click the “ID” link at the top of the column. This would order the rows as GUDMAP:1, GUDMAP:2, GUDMAP:3, etc. Clicking on the same column header again would sort the rows in reverse order of Gudmap ID.

Adding and removing columns
The display can be altered by adding or removing columns. Choose column name from the “Add Column”  or  “Remove Column”  dropdown menus displayed at the bottom of the page and click 'Go' button.  Up to eight columns can be displayed at the same time on the page. If the user tries to add more than 8 columns, an error message will inform .

Additional Links
In the “Microarray Browse All” page, for each submission there are links to additional pages relating to the submission. The Gudmap accession number (GUDMAP:ID) links to the relevant Submission Page.

The Sample column link to the corresponding entry in the GEO database.
