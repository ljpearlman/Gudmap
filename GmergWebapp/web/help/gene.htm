The Gene Details page.

This page contains information relating to a specific gene which is associated with an entry in the database

http://gudmap.hgu.mrc.ac.uk/gudmap/pages/gene.html?gene=Wnt4

http://gudmap.hgu.mrc.ac.uk/gudmap/pages/gene.html?gene=Zzz3

http://www.euregene.org/euregenedb/pages/gene.html?gene=Slco1a4

http://www.euregene.org/euregenedb/pages/gene.html?gene=Slc7a1

It is displayed by clicking on a gene symbol in the ISH Browse All page or in the gene list of the Microarray Submission Detail page.

Gene date is supplied for the following:

  • Symbol
  • Name
  • Synonyms
  • Chromosome locations and genome build it came from
  • External Links. Here the user will find a number of links to external databases which provide additional information about the selected gene. Users can link to MGI, OMIM, RefSeq, Ensembl, GO or Entrez online databases.

Cross reference ISH and Microarray entries.

From this page the user can also see all linked ISH and Microarray submissions in the database which relate to this gene.

Each of the :ID’s links back to either the ISH or Microarray Submission Page.
