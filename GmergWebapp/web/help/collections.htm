Collections

ISH and Microarray Data are saved in separate collections.

ISH Collection
From 'Browse ISH Data'

Users can make their own data collection of interest from the result pages by clicking the appropriate buttons at the foot of the page.

Select a number of rows in the query result list by clicking the checkbox in the first column followed by clicking the ‘Add To My Collection’ button.

This will take you to ‘My Collection’ page. From there further operations on the collection can be done:

  • In the My Collection page, delete the entries by selecting a number of rows in the result set and clicking ‘Delete From My Collection’
  • In the My Collection page, empty the result list by clicking Empty My Collection Button.

Columns in the ISH collection are:

  • Select
  • ID
  • Gene Symbol
  • Stage
  • Stage Given
  • Lab
  • Submission Date
  • Assay Type
  • Specimen Type
  • Thumbnails

By returning to the 'Browse ISH Data' page the user can subsequently:

  • Append to their collection by clicking Add To My Collection Button,
  • Replace the entries in the collection with the new result set by using Replace My Collection Button,
  • Make a comparison between the entries in the result set and the entries in the collection by clicking the ‘Get Difference With My Collection’ or ‘Get Intersection With My Collection’ buttons.

Microarray Collection
From 'Browse Microarray Data' -> Select a GUDMAP:ID -> Scroll down to the foot of the Gene List Section

By selecting from the dropdown menus, construct which type of operation you wish to perform on the Collection. Click GO.

Click the 'View My Gene Collection' to view the current contents of the collection. From here the user can

  • Delete the entries by selecting a number of rows in the result set and clicking ‘Delete From My Collection’
  • Empty the result list by clicking Empty My Collection Button.

The columns for Microarray Collections are:

  • Select
  • Gene Symbol (links to Gene Details Page)
  • Number of ISH Submissions Detected (links to Browse ISH format for that number of submissions for that gene)
  • Number of Microarray Submissions Detected (links to Browse Microarray format for that number of submissions for that gene)
